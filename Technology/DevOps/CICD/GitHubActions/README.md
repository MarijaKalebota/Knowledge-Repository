## GitHub Actions

```
GitHub Actions - CICD platform
Workflow - defined in the ".github/workflows", configurable automated process that will run one or more jobs when triggered
Event - specific activity in a repository that triggers a workflow run
Job - set of steps in a workflow that is executed on the same runner; a shell script that will be executed, or an action that will be run; ordered and dependent on each other; can share data with other steps
Runner - server that runs your workflows when they're triggered
Step - 
Action - custom application for the GitHub Actions platform that performs a complex but frequently repeated task
```
