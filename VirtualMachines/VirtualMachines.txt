## Virtual machines

A virtual machine is an operating system running in an existing operating system on the host machine's hardware .


### Virtualisation software

These software create a separate and secure environment on your host machine.
VMWare and Virtual box are among the well known virtualisation software.


### Installation

Tips and tricks at the bottom of: http://www.zemris.fer.hr/predmeti/os/

Download one of the virtualisation software.
Download Ubuntu or a similar Virtual Image.


#### VirtualBox

Create a Virtual Machine. Allocate enough RAM and hard disk memory. When running it the first time, select the
.iso image or the drive in which there is a DVD with the Virtual Image.

To share the clipboard/Drag'nDrop files, check the option under General Settings on the running VM.
